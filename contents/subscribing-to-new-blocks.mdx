The <ins>[EventEmitter API](https://docs.ethers.io/v5/single-page/#/v5/api/providers/provider/-%23-Provider--event-methods)</ins> allows applications to use an Observer Pattern to register callbacks for when various events occur.
We can use this to subscribe new blocks.

<CH.Spotlight>

<CH.Code>

```js subscribe.js
const { ethers } = require("ethers");

(async () => {
  const provider = ethers.getDefaultProvider();

  provider.on("block", async (blockNumber) => {
    console.log(blockNumber);
  });
})();
```
