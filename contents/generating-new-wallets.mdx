Generate a new Wallet with a random private key, generated from cryptographically secure entropy sources.
If the current environment does not have a secure entropy source, an error is thrown.

```js wallet.js focus=4
const { ethers } = require("ethers");

(async () => {
  const wallet = ethers.Wallet.createRandom();

  console.log("address", wallet.address);
  console.log("mnemonic", wallet.mnemonic);
  console.log("privateKey", wallet.privateKey);
})();
```

The mnemonic phrases are used to generate a private key.
Most modern cryptocurrency wallets implement <ins>[Bitcoin Improvement Proposal (BIP) 39](https://github.com/bitcoin/bips/blob/master/bip-0039.mediawiki)</ins>.

```js wallet.js focus=7
const { ethers } = require("ethers");

(async () => {
  const wallet = ethers.Wallet.createRandom();

  console.log("address", wallet.address);
  console.log("mnemonic", wallet.mnemonic);
  console.log("privateKey", wallet.privateKey);
})();
```
